Configuration Options

Use "tcp" as backend name with glip_new(). The following options are available:

tcp backend options
hostname the host to connect to, defaults to localhost. Both IPv4 and IPv6 addresses are supported, as well as FQDNs (such as example.com)
port the port to connect to, defaults to 23000
