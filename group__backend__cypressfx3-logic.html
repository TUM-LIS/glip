Detailed Description

The GLIP logic provides the interface between the GPIF-II of the Cypress FX3 chip and the user logic. Since the GPIF-II already provides a FIFO interface, the logic is rather minimal and consists mainly of the required clock-domain crossing infrastructure and down-/upscaling to/from 16 bit words.

The Verilog toplevel module glip_cypressfx3_toplevel implements the common GLIP logic interface. The following additional ports are needed for the external interface towards the GPIF-II of the FX3 chip. All signals with the _n suffix are active low, following the FX3 default. Be careful as the output polarity can be changed in the FX3 firmware!

Port Name Width Direction Description
fx3_pclk 1 OUT interface clock (supplied by FPGA)
fx3_dq 32 INOUT data bus
fx3_slcs_n 1 OUT chip select
fx3_slwr_n 1 OUT write strobe
fx3_sloe_n 1 OUT enable FX3 output on data bus
fx3_slrd_n 1 OUT read strobe
fx3_pktend_n 1 OUT packet end signal for short packet
fx3_a 2 OUT select GPIF-II socket
fx3_flaga_n 1 IN GPIF-II socket 0 buffer full
fx3_flagb_n 1 IN GPIF-II socket 0 buffer almost full
fx3_flagc_n 1 IN GPIF-II socket 3 buffer empty
fx3_flagd_n 1 IN GPIF-II socket 3 buffer almost empty
fx3_com_rst 1 IN communication reset
fx3_logic_rst 1 IN logic reset

USB Endpoint 1 (EP1) is the OUT endpoint, transferring data from the host to the device, as well as the IN endpoint, transferring data from the device to the host PC. P-port socket 3 of the GPIF-II is used for the USB OUT path, transferring data to the FPGA, P-port socket 0 is used for the USB IN path, accepting data from the FPGA. The data transfer between the USB endpoint and the P-ports is done via automatic DMA transfers.
