unsigned int glip_backend_ctx::target_word_width

target word width in bytes. 0 represents an invalid value

This value is read from the target at the begin of the transmission. In the hardware, it's configured in the WORD_WIDTH parameter passed to glip_jtag_toplevel. (Note that the hardware parameter is expressed in bit, while this variable is in bytes.)

unsigned int glip_backend_ctx::target_buffer_size

target buffer size in words. 0 represents an invalid value

This value is read from the target at the begin of the transmission. In the hardware, it's configured in the NUM_WORDS parameter passed to glip_jtag_toplevel. (Note that the configuration is done log2(size), i.e. the width of the size.)
